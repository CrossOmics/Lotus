fix: Add DEOptions and DEG analysis classes to cplearn namespace

Fix AttributeError when accessing cplearn.DEOptions in marker genes
analysis. The cplearn namespace object (_CplearnNamespace) was missing
DEG analysis related classes and functions.

Changes:
- Add DEOptions, DEAnalyzer, de_from_model, de_from_adata, and
  find_markers to _CplearnNamespace class attributes
- Add handling for these attributes in _module_getattr function to
  support both namespace and module-level access

This fixes the error "Marker genes analysis failed: cplearn has no
attribute 'DEOptions'" when running differential expression analysis.
